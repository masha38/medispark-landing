# 네이버 서치어드바이저 등록 가이드

## 등록할 사이트 주소

`https://masha38.github.io/medispark-landing/`

## 준비 완료된 항목

- 검색로봇 허용: `https://masha38.github.io/medispark-landing/robots.txt`
- 사이트맵: `https://masha38.github.io/medispark-landing/sitemap.xml`
- 대표 URL(canonical), 제목, 설명, OG 이미지, 파비콘 설정
- 네이버 검색로봇 `Yeti` 접근 허용

## 등록 순서

1. `https://searchadvisor.naver.com/`에 로그인합니다.
2. 웹마스터 도구에서 위 사이트 주소를 추가합니다.
3. 소유확인 방법에서 `HTML 태그`를 선택합니다.
4. 네이버가 제공하는 아래 형태의 태그 전체를 복사합니다.

```html
<meta name="naver-site-verification" content="발급된_확인코드" />
```

5. 복사한 태그를 개발자에게 전달합니다. `index.html`의 `<head>` 안에 넣고 재배포해야 합니다.
6. 배포 후 네이버 화면에서 `소유확인`을 누릅니다.
7. 소유확인이 끝나면 `요청 > 사이트맵 제출`에서 아래 주소를 제출합니다.

`https://masha38.github.io/medispark-landing/sitemap.xml`

8. `검증 > robots.txt`에서 수집 가능 여부를 확인하고 수집 요청을 누릅니다.
9. `요청 > 웹 페이지 수집`에서 메인 주소를 한 번 수집 요청합니다.

## 전달해야 할 마지막 자료

네이버에서 발급된 `naver-site-verification` 메타태그만 전달하면 소유확인 배포를 마칠 수 있습니다.

## 참고

- 소유확인 태그는 반드시 `<head>` 안에 있어야 합니다.
- GitHub Pages는 배포 반영에 잠시 시간이 걸릴 수 있으므로 새 버전이 열린 뒤 소유확인을 진행합니다.
- 사이트맵에는 검색에 노출할 메인 랜딩페이지만 포함했습니다.
